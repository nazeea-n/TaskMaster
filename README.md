**TaskMaster** is a simple web-based task manager that allows users to add, complete, and delete tasks. Tasks are saved in browser storage, so they persist on reload.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clear all tasks at once
- Task counter showing pending tasks
- Responsive and modern UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage

## How to Run
1. Download or clone the repository
2. Open `index.html` in a web browser
3. Start managing your tasks!
